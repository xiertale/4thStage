# Стилизация

## Глобальные стили

[src/styles/globals.tsx](src/styles/globals.tsx)

- css переменные (хранение цветовых схем и другие)
- шрифты
- общие стили html элементов таких как body, input, select, h1.., a и другие (эти стили будут применены ко всем элементам приложения)


## Стилизация компонентов

Для стилизации компонент применяются модульные стили
расширение .module.scss

Название классов в модульных стилей будут инкапсулированы внутри компонента, что обеспечивает изоляцию стилей.


### Общий шаблон 
В папке src/components/layout расположены компоненты шаблона такие как:
- Header - шапка
- Menu - меню
- Page - страница
- Main - основной контент
- Footer - подвал

Каждый компонент имеет файл модульных стилей
